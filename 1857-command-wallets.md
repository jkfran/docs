**Usage:** `juju wallets [options]`

**Summary:**

List wallets.

**Options:**

`-B, --no-browser-login (= false)`

Do not use web browser for authentication

`-c, --controller (= "")`

Controller to operate in

`--format (= tabular)`

Specify output format (`json|tabular`)

`-o, --output (= "")`

Specify an output file

**Details:**

List the available wallets.

**Examples:**

`   juju wallets`

**Aliases:**

list-wallets
